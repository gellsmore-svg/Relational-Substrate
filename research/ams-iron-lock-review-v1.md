# AMS Iron Lock Review v1

## Purpose

This note reviews whether the current AMS corpus now justifies leaving iron at:

```text
9.25/10 provisional upward revision
```

or moving it into a firmer locked position.

The question is no longer iron's identity.

That question is already settled strongly enough.

The question is now:

```text
does the current corpus justify firmer confidence discipline for the iron family?
```

## Central Claim

The central claim is:

> The current corpus justifies a firmer iron lock than before because the clearest structural gap, non-heme catalytic iron, has now been closed and the family identity survives chambered, relay, storage, toxicity, biomineral, and geological expressions; however, the present evidence still supports a cautious high-confidence lock better than a top-tier lock, because the geological end remains looser and the family still depends more visibly than the cleanest families on strong governance language.

More briefly:

```text
Iron is now firm enough to lock harder, but not yet clean enough to join the highest-confidence tier.
```

## The Strongest Reasons to Lock Iron More Firmly

There are four main reasons to remove the provisional status.

## 1. The Clearest Structural Gap Has Been Closed

The strongest prior reason for caution was explicit:

```text
non-heme catalytic iron still lacked a dedicated detailed map
```

That reason no longer stands.

The corpus now has a proper non-heme catalytic file, and that file shows that iron depth:

- survives outside porphyrin enclosure;
- still governs difficult oxygen activation;
- still requires local route discipline;
- still fails dangerously when uncoupled.

That is exactly the kind of new geometry that could have broken the family if the family had been overfitted.

It did not break it.

## 2. The Family Is Now Continuous Across Its Main Biological Roles

Iron no longer reads like several loosely associated domains.

It now reads as one family with several governed expressions:

```text
gas handling
-> catalytic oxidation
-> relay depth
-> storage/governance
-> toxicity boundary
```

That is a strong biological continuity arc.

## 3. The Failure Boundary Strengthens the Identity

Some families weaken when their pathology edge cases are inspected closely.

Iron strengthens.

The toxicity and ferroptotic boundary does not undermine the identity.

It confirms it:

```text
the same redox depth that is useful under governance becomes violently destructive when governance fails
```

That is strong family evidence, not peripheral noise.

## 4. The Family Already Integrates Well with the Core Machine Layer

Iron is no longer supported only by element-level route maps.

It now has strong machine-scale integration through:

- respiration and ATP synthesis;
- redox repair / oxidative boundary logic;
- cross-family machine synthesis.

That matters because the family is no longer standing on biochemical anecdotes alone.

It now participates in one of the best-constrained machine layers in the corpus.

## The Main Reasons Not to Push Iron Into the Top Tier

Even after the strengthening, there are still clear reasons not to move iron straight to the cleanest tier with `C`, `N`, `P`, and `O`.

## 1. Geological Breadth Is Still the Loosest End

The mineral-redox geological bridge remains valuable, but it is still looser than the strongest biochemical iron files.

The issue is not incoherence.

The issue is that:

- timescales are longer;
- route interpretation is less tightly bounded;
- the explanatory path is less direct than in heme, Fe-S, or non-heme systems.

That still argues for caution.

## 2. Governance Language Still Does More Work for Iron Than for the Cleanest Families

Iron is coherent, but it is coherent with a stronger rider than most families:

```text
iron depth must be embedded, stored, transported, governed, or mineralised to remain useful
```

That is not a defect.

It is part of the family identity.

But it does mean iron has more visible dependence on host conditions than the cleanest locked families.

## 3. Formal Calibration Still Lags the Strongest Comparative Confidence

The machine layer improved the broader corpus, but iron itself is still not formally calibrated across:

- heme versus non-heme depth;
- relay versus catalytic branch intensity;
- storage thresholds;
- ferroptotic escape thresholds.

That means the family is more strongly argued comparatively than formally.

