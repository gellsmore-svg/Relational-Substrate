# AMS Silicon Material Family Entry v1

## Purpose

This entry opens the silicon material-family workstream.

Silicon is the reference backbone for much of the current AMS material programme because it underlies:

- silica and silicate glass
- network-former / modifier comparisons
- soda-lime and aluminosilicate systems
- optical transmission through glass
- semiconductor crystals
- Si/SiO2 device interfaces
- biological silica in diatoms and sponges
- surface/interface route behaviour

The central AMS question is:

```text
How can one element support both rigid mineral network closure and high-precision electronic/interface behaviour?
```

## Source-Supported Working Facts

The source literature supports these working facts:

```text
Silica is a network former built from connected Si-O units.
Silicate glass has a three-dimensionally connected non-crystalline network without long-range periodic order.
Network modifiers such as Na and Ca disrupt or alter the silica network.
Glass structure and vibrational spectra are sensitive to composition, thermal history, stress, and surface treatment.
The Si/SiO2 interface is exceptionally important in semiconductor technology because of its high electrical quality.
Si/SiO2 interface bonding is not unique; intermediate silicon oxidation states occur at the interface.
Silicon nanocrystal / SiO2 systems show band-gap behaviour affected by confinement and surface/interface bonding.
Diatoms produce highly varied silica structures at nano- to microscale through coordinated biological silicification.
Silicification depends on silicon transport, silicic acid chemistry, cell-surface interactions, and biomolecular regulation.
```

## Primary Silicon Roles

### 1. Silica Network Former

Conventional role:

```text
SiO2 forms a connected network of Si-O units.
```

AMS mapping:

```text
role: S1 silica backbone former
route: R1
obstruction: O1
dominance: D1
route_grade: G_A
```

Interpretation:

Silica is the reference case for strong mineral network closure.

In pure silica glass, the network is non-crystalline but still coherent. It lacks long-range crystal periodicity while retaining connected material order.

AMS formula:

```text
local tetrahedral coordination
+ extended oxygen bridging
+ retained amorphous continuity
=> silica network closure
```

### 2. Silicate Modifier Baseline

Silicon also establishes the field in which modifiers operate.

AMS mapping:

```text
SiO2 backbone
=> M1 reference field

=> M2 weak-route opening

=> M3_Ca bridge stabilisation

MgO
=> M3_Mg selective gating

Al2O3
=> A1/A2/A4 topology switch depending compensation
```

This makes silicon the reference background for all previous glass work.

AMS reading:

```text
modifiers only have their AMS meaning relative to a network they modify.
```

### 3. Optical Transmission Medium

Silica and silicate glasses are central to optical transmission.

AMS mapping:

```text
role: S2 optical route medium
route: R1/R2 through coherent amorphous field
obstruction: O1, local O5/O6 if defects/scatterers dominate
dominance: D1/D4
route_grade: G_A/G_B when composition and surface are clean
```

Interpretation:

Glass transmits visible light well when the material has coherent amorphous continuity without large crystalline scattering domains or absorbing defect fields.

AMS optical reading:

```text
light travels through glass
because the substrate-facing material field admits coherent through-routes
without forcing strong route-breaking scattering at every local unit.
```

This connects silicon directly to the earlier interface programme:

```text
air -> glass
glass -> air
glass -> crystal
terminal layer
route release
surface defect fields
```

### 4. Semiconductor Crystal

Elemental silicon forms a crystalline semiconductor.

AMS mapping:

```text
role: S3 electronic lattice
route: R1
secondary_route: R2 under carrier motion / doping
obstruction: O1/O2
dominance: D1
route_grade: G_A
```

Interpretation:

In silicon electronics, the important closure is not only mechanical lattice stability. It is electronic admissibility: band structure, dopant states, surface passivation, and interface control.

AMS reading:

```text
crystal order
=> electronic route grammar

doping
=> controlled route perturbation

defect state
=> route obstruction or leakage path
```

### 5. Si/SiO2 Interface Platform

The Si/SiO2 interface is a major reason silicon became dominant in electronics.

Source anchor:

```text
The Si/SiO2 interface has unusually high electrical quality, though atomic bonding at the interface is not unique and intermediate oxidation states occur.
```

AMS mapping:

```text
role: S4 high-quality interface closure
route: R1/R3
obstruction: O1/O2
dominance: D1/D4
route_grade: G_A/G_B
```

Interpretation:

This interface is a strong AMS case because it requires two different closure systems to meet without producing too many electrically active defects.

AMS formula:

```text
Si crystal closure
+ SiO2 amorphous oxide closure
+ low-defect transition layer
=> functional electronic interface
```

This is directly relevant to the AMS terminal-layer and interface route work.

## Relation to Existing Families

### Silicon vs Sodium

```text
Si closes.
Na opens.
```

Silicon supplies the network field; sodium breaks or loosens it into workability and mobility.

### Silicon vs Calcium

```text
Si closes.
Ca bridges.
```

Calcium stabilises modified silicon networks and participates in mineral/biological scaffold closure.

### Silicon vs Magnesium

```text
Si closes.
Mg gates.
```

Magnesium imposes selective local constraint inside or near silicon networks.

### Silicon vs Aluminium

```text
Si closes neutrally.
Al closes conditionally.
```

Silicon forms the neutral tetrahedral backbone reference; aluminium can join the network when charge-compensated.

silicon material family: 9/10
