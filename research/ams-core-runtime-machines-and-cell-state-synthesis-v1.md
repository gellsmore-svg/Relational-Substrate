# AMS Core Runtime Machines and Cell State Synthesis v1

## Purpose

This note extends the runtime-machine comparison to the level of whole-cell state.

The two strongest machine-scale passes already mapped are:

- respiration plus ATP synthesis;
- ribosome plus translation.

The question now is:

```text
how do these machines couple into cell-level coherence?
```

The locked family identities remain:

```text
Carbon carries molecular architecture.
Nitrogen carries biological availability and recognition.
Phosphorus carries transferable biological order.
Sulfur carries controllable chemical reactivity.
Iron carries redox depth.
Oxygen carries polarity and oxidative power.
```

## Central Claim

The central claim is:

> Cell state is the coupled condition of architecture, readability, transferable order, governed reactivity, redox depth, and polarity/oxidative field. A viable cell is not merely a bag containing working subsystems. It is a maintained runtime coherence in which respiration, translation, membrane asymmetry, repair, and growth remain mutually supportable.

## Coupling Logic

These six states are not independent.

### Respiration -> ATP -> Translation

```text
redox depth
-> oxygen terminal closure
-> proton field
-> ATP
-> translation cycle progression
-> protein renewal
```

This is the most obvious coupling chain.

If respiration weakens:

- phosphorus runtime order falls;
- translation slows or becomes error-prone;
- architectural renewal weakens;
- repair systems lose support.

### Translation -> Machine Renewal

```text
readable sequence
-> protein architecture
-> repair enzymes, transporters, channels, redox proteins, ribosomal parts
```

Translation is therefore not just output generation. It is the renewal route for the rest of the runtime machinery.

If translation coherence fails:

- respiratory machinery cannot be renewed correctly;
- membrane traffic and repair weaken;
- redox governance degrades;
- cell state drifts toward failure even if ATP is still temporarily available.

### Membrane State as Shared Host

```text
carbon boundary architecture
+
oxygen-facing phosphorus order
=
host for gradients, transport, signalling, and separation
```

The membrane is where:

- proton asymmetry persists;
- transport occurs;
- compartments are distinguished;
- viability thresholds often become visible first.

If membrane coherence fails:

- respiration uncouples;
- transport collapses;
- signalling degrades;
- the cell loses inside/outside runtime distinction.

### Redox Governance as Safety Layer

```text
iron depth + oxygen power
must be held inside sulfur-mediated repair and buffering
```

Without this layer:

- ROS rises;
- lipid peroxidation spreads;
- Fe-S systems degrade;
- proteins and nucleic acids accumulate damage;
- ATP and translation eventually fail secondarily.

## Three Major Cell States

The synthesis currently supports three broad cell-state classes.

### 1. Coherent Growth State

Profile:

```text
architecture maintained
readability high
ATP and phosphate order available
redox depth governed
oxygen closure productive
repair mesh keeping pace
```

Interpretation:

The cell is not merely alive. It can:

- grow;
- divide;
- repair;
- differentiate;
- maintain selective transport;
- sustain machine renewal.

### 2. Stress Compensation State

Profile:

```text
architecture partly strained
readability still mostly intact
ATP pressure rising
redox governance strained
ROS elevated but buffered
translation and repair prioritised
```

Interpretation:

The cell is still viable but now spends more of its runtime budget on:

- correction;
- triage;
- repair;
- damage limitation;
- selective shutdown or slowdown.

This state is important because it shows that viability is graded, not binary.

### 3. Failure State

Profile:

```text
architecture decouples from renewal
readability loses fidelity
ATP capture falls
redox depth escapes governance
oxygen fragmentation outruns repair
membrane or information coherence crosses threshold
```

Interpretation:

At this point the cell is no longer maintaining a stable runtime loop.

Different failure modes may dominate:

- oxidative collapse;
- ferroptotic membrane failure;
- energy starvation;
- translation collapse;
- proteostatic overload;
- information damage accumulation.
