# AMS Severe Ecology Patterns Comparison v1

## Purpose

This file compares the main severe ecology patterns now present in the corpus.

Recent work has developed several distinct high-danger states:

- acute shock collapse
- chronic decay path
- hollow-support condition
- mixed burden spiral
- containment-loss cascade

The current question is:

```text
are these genuinely distinct ecology patterns,
or just different descriptions of severe failure?
```

This file argues that they are genuinely distinct and compares them directly.

## Central Claim

The central claim is:

> Severe living decline is not one thing in larger or smaller degree. The current AMS ecology layer now supports several distinct severe patterns, each with its own burden organisation, support condition, reserve profile, synchrony shape, and death-approach logic. This matters because the strongest life-boundary and recovery claims depend on distinguishing these patterns rather than compressing them into one generic collapse category.

More briefly:

```text
Badly failing ecologies do not all fail the same way.
```

## The Five Main Severe Patterns

The strongest current severe ecology patterns are:

1. acute shock collapse
2. chronic decay path
3. hollow-support condition
4. containment-loss cascade
5. mixed burden spiral

These can now be compared directly.

## 1. Acute Shock Collapse

### Core shape

```text
fast destabilisation under high acute burden
```

### Typical ecology signature

- acute burden very high
- reserve rapidly spent
- support overwhelmed quickly
- synchrony rises sharply
- cascade pressure becomes dominant fast

### Defining feature

The defining feature is:

```text
the ecology is driven over faster than it can redistribute burden meaningfully
```

### Best current examples

- respiratory crash
- acute energetic collapse
- abrupt severe oxidative destabilisation

## 2. Chronic Decay Path

### Core shape

```text
slower deterioration under cumulative burden
```

### Typical ecology signature

- acute burden moderate or lower
- cumulative burden high and rising
- reserve erodes gradually
- support thins slowly
- synchrony increases over longer time scales

### Defining feature

The defining feature is:

```text
future recoverability is being consumed before present-tense coherence fully collapses
```

### Best current examples

- translation-heavy renewal debt
- chronic architecture erosion
- long-run repair exhaustion

## 3. Hollow-Support Condition

### Core shape

```text
support routes remain structurally present but functionally weak
```

### Typical ecology signature

- local function still visible
- structural support still present
- effective support weak
- reserve shallow
- synchrony high enough that support no longer restores well

### Defining feature

The defining feature is:

```text
the ecology still looks connected, but no longer helps itself effectively
```

### Best current examples

- severe but still active compensation states
- near-death states with local function preserved
- repair-active but support-thin ecologies

## 4. Containment-Loss Cascade

### Core shape

```text
bounded separability fails and local burdens spread aggressively
```

### Typical ecology signature

- boundary weakness or retention loss
- abrupt rise in synchrony
- support routes become less usable
- cascade pressure spikes sharply
- reserve is consumed quickly by spread rather than repair alone

### Defining feature

The defining feature is:

```text
the ecology stops being able to keep local trouble local
```

### Best current examples

- boundary-collapse states
- severe membrane/gradient loss
- compartment-destabilised crisis

## 5. Mixed Burden Spiral

### Core shape

```text
recovery attempts remain real but become net ecological loss
```

### Typical ecology signature

- acute and cumulative burden both significant
- synchrony persistently high
- reserve baseline drifts downward
- support hollows cycle by cycle
- positive loop still acts locally but negative loop dominates globally

### Defining feature

The defining feature is:

```text
the system keeps trying to recover without restoring future recoverability
```

### Best current examples

- coupled respiration-renewal decline
- burden spirals with partial restabilisation
- severe states that oscillate near viability threshold

## What Distinguishes These Patterns Most Sharply

The comparison shows that five variables matter most in distinguishing severe ecology patterns:

1. acute versus cumulative burden dominance
2. reserve consumption speed
3. support effectiveness versus support topology
4. synchrony rise shape
5. whether positive loops are absent, overwhelmed, or still locally active

This is a major gain because it shows the corpus now has real discriminators rather than only descriptive labels.

## The Strongest Distinctions

The comparison clarifies several especially important distinctions.

## 1. Acute Shock Is Not the Same as Mixed Spiral

Both can end badly and both can involve high cascade pressure.

But:

- acute shock is dominated by rapid overrun
- mixed spiral is dominated by repeated failed recovery

That is a real ontological difference.

## 2. Chronic Decay Is Not the Same as Hollow Support

Both may keep the system visibly active for some time.

But:

- chronic decay is mainly about cumulative burden accumulation
- hollow support is mainly about supportive relations becoming ineffective even while some structure persists

That is a different ecological shape.

## 3. Containment Loss Is Ecologically Special

Containment-loss cascade is not just another version of severe load.

Its distinctive role is:

```text
destroying separability
```

That makes it one of the clearest synchrony accelerants in the whole ontology.

## 4. Mixed Spiral Is the Strongest False-Recovery Pattern

This may be the most important distinction in the whole file.

The mixed spiral explains why:

- repair can be active;
- compensation can be visible;
- some recoveries can occur;

and yet:

```text
death approach can still be deepening
```

That is one of the strongest current anti-naive-recovery points in the corpus.

## 5. Hollow Support and Mixed Spiral Are Closely Related but Not Identical

This distinction matters.

Hollow support is a condition:

```text
support is no longer effectively restorative
```

Mixed spiral is a process:

```text
recovery cycles continue while hollow support helps drive net decline
```

That means hollow support often appears inside mixed spirals, but the two should not be collapsed into one concept.
