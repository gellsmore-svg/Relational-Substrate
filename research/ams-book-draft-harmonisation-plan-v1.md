## The Four Main Harmonisation Goals

The current draft should now be edited toward four main goals.

## 1. Seam Quality

Each chapter should feel like it hands naturally into the next.

This means:

- the end of each chapter should create real forward pressure;
- the opening of the next chapter should pick that pressure up cleanly;
- no chapter should feel like a reset.

This is the first major editorial goal.

## 2. Repetition Control

Some phrases and argument moves now recur because the chapters were drafted from strong packets with overlapping conceptual cores.

That is natural, but it now needs discipline.

The goal is not zero repetition.

The goal is:

```text
strategic repetition of the core grammar,
not repeated re-explanation of the same move at full length
```

## 3. Density Control

The book should feel cumulative, not swollen.

This means:

- technical support should stay beneath the prose where possible;
- the strongest conceptual lines should carry the narrative;
- not every chapter needs to carry the full weight of the research layer.

## 4. Voice Consistency

The draft is already fairly coherent in tone, but the harmonisation pass should ensure:

- consistent register;
- similar sentence density;
- stable chapter pacing;
- similar opening and closing strength across chapters.

The manuscript should sound like one author making one cumulative case.

## The Main Repetition Risks

The current draft should be specifically checked for over-repetition in five areas.

## 1. Created Bounded Continuity

This phrase and its close variants are central, but should not be redefined from scratch in every chapter.

Use:

- full explanation early;
- lighter callback later.

## 2. “Same Damaged Continuity”

This language is essential in the corruption/redemption arc, but can become too repetitive if used at full intensity in both chapters and again in the resurrection chapter.

It should be distributed rather than duplicated.

## 3. “Not X, Not Y, but Z” Structures

The draft uses these contrasts well, but a harmonisation pass should check whether this rhetorical pattern is appearing too frequently in the same form.

Variety would help.

## 4. Recoverability Language

Chapters 4, 5, 6, 7, and 8 all benefit from recoverability logic, but the prose should distinguish:

- life-level recoverability;
- personal continuity;
- restorative redemption;
- resurrection continuity

without sounding like one sentence template reused repeatedly.

## 5. AI Boundary Echoes

The AI/life/person distinction is important, but once Chapter 5 is drafted it should be checked to ensure the AI contrast remains proportionate and does not echo too often in later chapters where it is not central.
