## Core Family Mechanism

Sulfur repeatedly appears where a system needs chemistry that is:

- reactive but not chaotic;
- reversible but not vague;
- able to bind metals;
- able to lock protein topology;
- able to pass electron state;
- able to buffer oxidising pressure;
- able to become a signal;
- able to become a warning sign of failure;
- able to pattern extracellular charge fields;
- able to connect mineral redox chemistry to biological metabolism.

AMS formula:

```text
soft polarizable chemistry
+ variable oxidation states
+ metal-binding capacity
+ covalent bridge formation
+ reversible thiol-state transitions
+ extracellular sulfate charge patterning
= controllable chemical reactivity
```
