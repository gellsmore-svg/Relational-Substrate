# AMS C N P S Fe O Molecule Runtime Bridge v1

## Purpose

This note bridges the locked core element families:

```text
Carbon
Nitrogen
Phosphorus
Sulfur
Iron
Oxygen
```

to representative molecule and complex classes.

The question is no longer "what does each element-family do in isolation?" but:

```text
how do the locked family identities combine inside real molecular systems?
```

## Locked Family Basis

The current locked family identities are:

```text
Carbon carries molecular architecture.
Nitrogen carries biological availability and recognition.
Phosphorus carries transferable biological order.
Sulfur carries controllable chemical reactivity.
Iron carries redox depth.
Oxygen carries polarity and oxidative power.
```

This file tests whether those compressed sentences remain sufficient at molecule scale.

```text

hydrogen partner
```

```text
oxygen supplies polarity and hydrogen-bond ordering
```

Compressed rule:

```text
water is oxygen's field expression.
```


These failure cases are important because they show the family identities are not only descriptive of healthy molecules. They also predict the character of breakdown.
