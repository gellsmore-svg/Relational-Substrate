# AMS Defect, Hysteresis, and Memory Corpus v1

## Purpose

This corpus gathers cases where a whole carries retained history.

These are high-pressure cases because they test whether AMS can explain:

- not only what a thing is
- but how prior strain, alignment, forcing, or damage remains active in what it now does

That means this corpus is about:

- defects
- hysteresis
- path dependence
- retained structural memory

## Governing Question

The key question is:

- when does prior history become part of the current ontological state rather than merely background narrative?

AMS should say that in many cases, prior history is not external commentary. It is part of the present state of the whole.

## Case 1: Magnetic Hysteresis

### Why This Case Matters

Magnetic hysteresis is one of the clearest cases where prior alignment affects current behaviour.

### Observed

Magnetic systems can:

- retain alignment history
- resist reversal
- show path-dependent behaviour

### AMS interpretation

This is a core test for whether AMS can treat retained alignment as real structural state rather than as an accidental side note.

- `Memory type`: alignment memory
- `State significance`: very high
- `Failure / transition mode`: coercive reversal, thermal disorder, domain disruption
- `Higher-order significance`: high

### Confidence

- `Observed`: strong
- `Strong AMS inference`: strong
- `Confidence score`: `8/10`

## Case 5: Biological Conformational Memory

### Why This Case Matters

Biological assemblies often depend on prior folding, binding, or conformational state.

### Observed

Biological systems can show:

- state-dependent function
- altered response after prior binding or structural change
- failure when conformational order is lost

### AMS interpretation

This is a very important bridge case between materials and living systems. It suggests that memory can be real at structural and functional levels without requiring a new ontology.

- `Memory type`: conformational and functional history
- `State significance`: high
- `Failure / transition mode`: misfolding, denaturation, functional loss
- `Higher-order significance`: very high

### Confidence

- `Observed`: strong
- `Conjectural AMS extension`: moderate to strong
- `Confidence score`: `7/10`
