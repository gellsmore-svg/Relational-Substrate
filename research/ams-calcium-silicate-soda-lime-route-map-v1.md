# AMS Calcium Silicate and Soda-Lime Route Map v1

## Purpose

This file maps calcium's role in calcium silicate and soda-lime silicate glass systems.

It completes the first calcium triad:

```text
calcium carbonate
=> polymorph closure geometry

hydroxyapatite / collagen
=> T1B-to-T1C nested biological closure

calcium silicate / soda-lime glass
=> non-biological T1B modifier-stabiliser route control
```

The central question is:

```text
How does CaO stabilise a silicate network that Na2O has made more workable but also more soluble and mobile?
```

## Source-Supported Working Facts

The source literature supports the following working facts:

```text
soda-lime-silicate glass is primarily SiO2, Na2O, and CaO
SiO2 is the network former
Na and Ca act as network modifiers
bridging oxygens connect neighbouring silicon atoms
non-bridging oxygens are associated with modifier ions such as Na or Ca
Na2O lowers melting/working temperature but increases solubility if not stabilised
CaO/lime improves chemical durability and stabilises the glass
in calcium aluminosilicate glasses, CaO acts as a network modifier and increases non-bridging oxygen content
soda-lime-silicate glass is widely used because it combines workability, transparency, hardness, and chemical durability
```

## AMS Core Claim

Calcium in soda-lime silicate glass is not merely another network breaker.

AMS reading:

```text
Na2O opens the silicate network for workability.
CaO stabilises the opened network enough for durable retained form.
```

This creates a three-role structure:

```text
SiO2
=> network former / backbone closure

Na2O
=> fluxing modifier / weak-route opener

CaO
=> bridge-stabilising modifier / route regulator
```

## Route Roles

### Silica Backbone

AMS role:

```text
M1: backbone closure
```

Route profile:

```text
route: R1
obstruction: O1
dominance: D1
grade: G_A
```

Interpretation:

Silica provides the primary network order. Pure silica has strong high-temperature stability and high viscosity, but it is difficult to process.

AMS formula:

```text
SiO2 backbone
=> strong R1 closure
=> high retained order
=> low workability
```

### Sodium Modifier

AMS role:

```text
M2: distributed weak-route opener
```

Route profile:

```text
route: R2
obstruction: O5
dominance: D3 pressure
grade: G_B/G_C depending on connectivity
```

Interpretation:

Sodium lowers working temperature and opens the network by creating non-bridging oxygen sites. It increases mobility and workability but can make the glass too soluble or weak if not stabilised.

AMS formula:

```text
Na2O
=> non-bridging oxygen increase
=> lower viscosity / easier flow
=> weak-route diffusion pressure
```

### Calcium Modifier-Stabiliser

AMS role:

```text
M3_Ca: bridge-stabilising modifier
```

Route profile:

```text
route: R1/R2
secondary_route: R3 under high local constraint
obstruction: O1/O2
dominance: D1/D4
grade: G_B/G_A in balanced systems
```

Interpretation:

Calcium also modifies the network, but its bivalent character allows it to stabilise modifier-opened structures more strongly than sodium alone.

AMS formula:

```text
CaO
=> modifier site participation
=> bivalent charge stabilisation
=> weak-route regulation
=> increased durability relative to Na-only silicate
```

Calcium is therefore a route regulator:

```text
not pure backbone
not pure diffusor
but stabilising bridge across opened regions
```

## Soda-Lime Glass Whole-Field Model

Whole-field AMS profile:

```text
soda_lime_silicate
= M1 silica backbone
+ M2 sodium weak-route opener
+ M3_Ca bridge-stabilising modifier
```

Route result:

```text
primary_route: R1/R2
secondary_route: R3 at local gates
primary_obstruction: O1
secondary_obstruction: O2/O5
dominance: D1/D4
route_grade: G_A/G_B
```

This is why soda-lime glass can be:

```text
workable
transparent
hard enough
chemically durable enough
industrially cheap
```

It is not maximally coherent like pure silica, but it is practically coherent across multiple constraints.

## Calcium vs Sodium in Silicate Glass

| Feature | Sodium role | Calcium role |
|---|---|---|
| charge | +1 | +2 |
| AMS modifier type | weak-route opener | bridge-stabilising modifier |
| route | `R2` | `R1/R2` |
| obstruction | `O5` | `O1/O2` |
| dominance pressure | `D3` | `D1/D4` |
| material effect | lowers viscosity / raises mobility | improves durability / stabilises modified network |
| risk if dominant | excessive solubility / diffusion | over-stabilisation or crystallisation tendency depending composition |

## Calcium vs Magnesium in Silicate Glass

| Feature | Magnesium role | Calcium role |
|---|---|---|
| ionic character | smaller, higher field strength | larger, lower field strength |
| AMS type | selective gate | bridge-stabilising modifier |
| route | `R1/R3` | `R1/R2` |
| obstruction | `O1/O2` | `O1/O2`, less sharply gated |
| mixed behavior | stronger interruption | broader stabilisation |
| biological/material compatibility | strong but more locally restrictive | broad scaffold compatibility |

Calcium's value is not that it is stronger than magnesium. Its value is that it is more broadly accommodating.

AMS reading:

```text
Mg gates.
Ca bridges.
Na opens.
Si closes.
```

## Vorton Interpretation

Soda-lime silicate glass contains a competing but stable vorton field:

```text
silica closure vortons
+ sodium-opened weak-route disruptions
+ calcium bridge-stabilisation vortons
=> durable amorphous retained form
```

Candidate vorton roles:

```text
SiO2
=> long-range amorphous network closure

Na2O
=> route-loosening defects / mobility vortons

CaO
=> bridge vortons that damp runaway diffusion
```

This makes soda-lime glass a useful AMS case because it is not a perfect lattice and not a random liquid. It is a retained amorphous closure field with tuned modifier competition.

## Interface and Optical Implications

Soda-lime glass is optically useful because its amorphous network can transmit visible light without large-scale scattering from crystallites.

AMS interface reading:

```text
amorphous retained closure
=> no large periodic crystal boundary scattering
=> light route can transmit through bulk
```

But surface/interface response depends on local modifier distribution.

Expected local exposure:

```text
silica-rich surface
=> cleaner R1 release

sodium-rich surface
=> more R2/O5 diffuse release, higher leaching vulnerability

calcium-rich stabilised region
=> R1/R2 regulated release, improved durability
```

This reconnects calcium silicate modelling to the earlier optical/interface work:

```text
glass transmission
depends on retained amorphous coherence

surface durability
depends on whether weak-route modifiers are stabilised

reflection/release behavior
depends on terminal-layer route structure
```

## Durability Implication

AMS durability claim:

```text
Na-only silicate
=> too much M2/O5 weak-route pressure
=> water-solubility risk

soda-lime silicate
=> Ca bridge-stabilisation reduces uncontrolled weak-path dissolution
=> improved chemical durability
```

Route form:

```text
R2/O5
+ Ca M3_Ca bridge
=> R1/R2 with O1/O2 gating
```

This is not saying calcium removes all vulnerability. It changes the route field from open diffusion toward regulated retained form.

## Viscosity and Workability Implication

AMS viscosity claim:

```text
SiO2 alone
=> too high closure resistance
=> poor workability

Na2O addition
=> route loosening
=> lower working temperature

CaO addition
=> retained stability without returning fully to pure-silica stiffness
```

Whole-field balance:

```text
workability
= enough R2 admission

durability
= enough R1/R2 calcium stabilisation
```

Soda-lime glass is successful because it occupies the usable middle:

```text
not too closed to work
not too open to persist
```

## Calcium Silicate Subcase

Calcium silicate systems, including wollastonite-like phases and calcium silicate glass/ceramic fields, are useful for testing calcium's bridge-stabilising role without sodium dominance.

AMS profile:

```text
calcium_silicate
primary_route: R1/R2
primary_obstruction: O1/O2
dominance: D1/D4
route_grade: G_B/G_A depending crystallinity
```

Crystalline calcium silicate:

```text
more R1/D1
```

Amorphous calcium silicate glass:

```text
more R1/R2 and D4
```

Bioactive calcium silicate/phosphosilicate systems add a T1C-adjacent layer because dissolution and apatite formation can become biologically relevant.
