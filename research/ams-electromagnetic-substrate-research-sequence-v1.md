# AMS Electromagnetic Substrate Research Sequence v1

## Purpose

This roadmap sequences the next research work around:

- magnetic behaviour through materials;
- electrical boundedness along conductive paths;
- darkness as ordered substrate ground;
- vorton generation and closure;
- optical route work already begun from glass/light cases.

The aim is coherence, not proof of an invisible substrate. AMS cannot prove the substrate by direct inspection. It can only become more or less coherent by showing whether one disciplined grammar can organise known behaviours without category drift, hidden exceptions, or arbitrary special pleading.

## Governing Principle

The governing principle is:

```text
calibrate simple material cases before explaining composite behaviour
```

The sequence should therefore move from:

```text
elemental roles
-> simple material regimes
-> composite substances
-> route and boundary behaviour
-> integrated electromagnetic grammar
-> darkness / vorton generation synthesis
```

This keeps later claims dependent on earlier constraints rather than on free metaphor.

## Research Sequence

## Phase 1 - Elemental Magnetic and Electrical Response Roles

### Purpose

Before studying composites such as wood, glass, water, or biological tissue, establish the elemental response grammar.

This phase asks:

```text
what kinds of elemental or simple-material roles support magnetic response,
electrical conduction,
dielectric delay,
reflection,
and insulation?
```

### Core cases

- iron;
- copper;
- aluminium;
- silicon;
- oxygen;
- carbon;
- hydrogen;
- calcium;
- sodium;
- magnesium.

### Deliverables

```text
ams-elemental-electromagnetic-response-table-v1.md
ams-iron-copper-aluminium-magnetic-electrical-contrast-v1.md
ams-dielectric-elements-and-network-formers-v1.md
```

### Why first

Composite materials are not primitive explanatory units. Wood, glass, water, and tissue inherit behaviour from elemental roles, bonding regimes, microstructure, moisture, impurities, and boundary organisation.

The elemental table should not claim that elements alone explain everything. It should define first-order tendencies that later composite files must respect.

## Coherence Tests

Each file should be judged by these tests.

### Test 1 - Does it reduce special pleading?

The same route variables should explain multiple cases without inventing new terms for each material.

### Test 2 - Does it preserve known physics?

AMS should reinterpret measured behaviours, not deny them.

### Test 3 - Does it distinguish behaviour types?

Transmission, reflection, conduction, dielectric delay, magnetic alignment, shielding, absorption, and scattering must not collapse into one vague "substrate response."

### Test 4 - Does it improve comparison?

A good file should explain why two materials differ, not merely describe one material elegantly.

### Test 5 - Does it avoid overclaiming proof?

No file should claim that AMS has proven the substrate. The strongest allowable claim is coherence under disciplined reinterpretation.
