# AMS Aluminium Material Family Entry v1

## Purpose

This entry opens the aluminium material-family workstream.

Aluminium is now necessary because it has appeared repeatedly in:

- Na/Mg aluminosilicate transport
- calcium aluminosilicate glass structure
- soda-lime and aluminosilicate strengthening
- charge-compensation gates
- bridging oxygen / non-bridging oxygen balance
- glass durability, stiffness, ion exchange, and topology

The central AMS question is:

```text
How can aluminium act as a network former, charge-coupled gate, or modifier depending on composition?
```

## Source-Supported Working Facts

The source literature supports the following working facts:

```text
Al3+ can substitute for Si4+ in tetrahedral network sites as [AlO4]-
[AlO4]- requires charge compensation by alkali or alkaline-earth cations
charge-compensated AlO4 can increase network connectivity
Al2O3 can improve mechanical and chemical properties in aluminosilicate glasses
in peralkaline compositions, Al is mainly tetrahedral and network-forming
in peraluminous or modifier-limited compositions, higher-coordination Al[V] or Al[VI] and oxygen triclusters may appear
network-modifying cations can either charge-compensate AlO4 or create non-bridging oxygens
Al/Na, Al/Ca, and Al/modifier ratios strongly affect network topology, ion mobility, durability, and stiffness
```

## AMS Family Classification

```text
Element family: aluminium
Primary ion in glass context: Al3+
Primary AMS role: conditional network former / charge-coupled gate
Secondary AMS roles: topology stiffener, modifier competitor, ion-exchange regulator, durability enhancer
Main material domains: aluminosilicate glasses, calcium aluminosilicates, sodium aluminosilicates, glass-ceramics, clays/minerals
```

Aluminium is not a simple `M1`, `M2`, or `M3` species.

It is conditional:

```text
properly charge-compensated Al[IV]
=> M1-like backbone participation

modifier-limited or high-Al condition
=> higher-coordination Al / tricluster behavior
=> gate or topology complication

excess modifiers
=> Al competes with NBO formation for modifier charge
```

## Core AMS Claim

Aluminium is a topology switch.

AMS reading:

```text
Al does not merely add strength.
It changes which routes are possible by changing charge-compensated network topology.
```

The key mechanism:

```text
SiO4 network
+ AlO4- substitution
+ charge-compensating cation
=> altered network connectivity
=> changed route grammar
```

This makes aluminium a gate/backbone hybrid:

```text
Al[IV] with compensation
=> joins the backbone

Al without proper compensation
=> creates structural complication, higher coordination, or local gate strain
```

## Primary Aluminium Roles

### 1. Network Former by Charge Compensation

When aluminium enters tetrahedral coordination as `AlO4-`, it can become part of the silicate-like network.

AMS mapping:

```text
Al[IV]O4-
+ Na+/K+/Ca2+/Mg2+ charge compensation
=> network-forming participation
=> M1_Al
```

Route profile:

```text
primary_route: R1
secondary_route: R3 if compensation is strained
primary_obstruction: O1
secondary_obstruction: O2
dominance: D1/D4
route_grade: G_A/G_B
```

Interpretation:

Aluminium can increase polymerization and reduce non-bridging oxygen content when the modifier charge is used for compensation rather than network breaking.

AMS formula:

```text
modifier charge used for AlO4 compensation
=> less charge available for NBO creation
=> stronger network connectivity
=> higher R1 dominance
```

### 2. Charge-Coupled Gate

Aluminium creates a gate because it cannot simply enter the Si network neutrally.

It requires compensation:

```text
Al3+ replacing Si4+
=> one negative charge on AlO4-
=> cation compensation required
```

AMS mapping:

```text
AlO4- site
=> M1/M3 gate node
```

Route profile:

```text
primary_route: R1/R3
primary_obstruction: O1/O2
dominance: D4 when compensation competition is active
route_grade: G_B/G_C under strained compensation
```

Interpretation:

Aluminium is not only a backbone element. It creates charge-dependent route admission.

This explains why cations such as Na, Ca, and Mg may behave differently depending on whether they are:

```text
charge compensators
or
network modifiers
or
free/excess oxides
```

### 3. NBO Suppressor or NBO Competitor

In sodium silicate glass, sodium can create non-bridging oxygens and open weak routes.

With aluminium present, sodium may instead compensate `AlO4-`.

AMS mapping:

```text
Na as NBO modifier
=> M2 / R2 / O5

Na as AlO4 compensator
=> M1/M3 gate support / R1
```

This is a major AMS discriminator:

```text
same sodium ion
different local role
different route class
```

Aluminium therefore changes sodium from:

```text
weak-route opener
```

into:

```text
backbone charge-support participant
```

depending on composition.

### 4. Higher-Coordination Complication

When composition prevents all aluminium from remaining tetrahedral and properly charge-compensated, higher-coordination Al can appear:

```text
Al[V]
Al[VI]
oxygen triclusters
```

AMS mapping:

```text
higher coordination Al
=> topology complication
=> gate strain
=> local D4
```

Route profile:

```text
route: R3/R4
obstruction: O2/O4
dominance: D4
grade: G_B/G_C
```

This does not automatically mean failure. It means the network is solving charge/topology constraints through a different local geometry.

## Aluminium in Na/Mg Aluminosilicate Transport

The previous Na/Mg work identified the transport refinement:

```text
Na mobility
= M2 route potential
x M1 network admission
x M3 gate condition
x M4 mixed-junction interference
```

Aluminium supplies much of the `M1/M3` gate condition.

AMS reading:

```text
AlO4 charge-compensation sites
=> gates through which Na mobility is admitted, redirected, or suppressed
```

This explains why Na mobility in aluminosilicate systems cannot be reduced to sodium content alone.

Route map:

```text
Na near NBO
=> R2/O5/D3 pressure

Na compensating AlO4
=> R1/O1/D1 pressure

Mg/Ca bridging AlO4 sites
=> R3/O2/D4 pressure

mixed local compensation
=> M4 junction behavior
```

## Aluminium in Calcium Aluminosilicates

In calcium aluminosilicate systems, calcium can act as a charge balancer for aluminium.

AMS mapping:

```text
Ca2+
+ AlO4-
=> bridge-compensated backbone site
```

This makes the calcium/aluminium pair important:

```text
Ca bridges.
Al switches topology.
Together they regulate network polymerization and reactivity.
```

Possible regimes:

```text
Ca/Al balanced
=> charge-compensated Al[IV]
=> R1/D1

Ca excess
=> Ca modifier behavior and NBO creation
=> R1/R2/D4

Al excess or low silica
=> Al[V]/Al[VI] or tricluster behavior
=> R3/R4/D4
```

## Aluminium in Chemical Strengthening

Aluminosilicate glasses are important in chemically strengthened cover glass systems.

Source anchors indicate that Al2O3 and MgO can improve network connectivity, mechanical properties, ion-exchange behavior, hardness, and surface compressive stress.

AMS mapping:

```text
Al-enhanced network
=> stronger backbone/gate structure
=> controlled Na/K ion exchange
=> surface compression can be retained
```

Route interpretation:

```text
ion exchange without network collapse
requires
strong R1 backbone
+ controlled R2 exchange route
+ gate compatibility
```

This is a high-value future subcase because it ties aluminium to:

- mobile ion exchange
- surface stress
- crack resistance
- device cover glass
- terminal-layer interface behavior

## Vorton Interpretation

Aluminium introduces charge-coupled closure.

Candidate vorton behavior:

```text
AlO4- tetrahedron
=> incomplete closure without compensator

AlO4- + cation
=> charge-closed network site

Al-rich / modifier-poor region
=> strained or higher-coordination closure
```

AMS reading:

```text
aluminium vortons are not self-sufficient.
They require compensating relation to complete stable closure in the silicate network.
```

This is why aluminium is a strong test of relational ontology:

```text
identity depends on co-ordination and compensation
not isolated elemental presence.
```

## Initial Aluminium Family Score

```text
aluminium material family: 8/10
confidence: moderate-high
```

Reason:

Aluminium is a strong AMS family because:

- it is composition-sensitive in a way AMS can model
- it changes network topology rather than merely adding a property
- it explains charge-compensation gates
- it regulates NBO formation
- it affects durability, stiffness, ion exchange, and transport
- it clarifies prior Na/Mg and Ca workstreams

It does not start above 8 because:

- aluminium biological roles are not yet mapped
- clays and natural aluminosilicate minerals are not yet mapped
- surface/interface and crack-resistance roles need detail
- aluminium toxicity/biological incompatibility boundaries need separate handling
- composition-specific Al/Na and Al/Ca maps are still missing
