# AMS Vorton Ontology Lock Definition Sheet v1

## Purpose

This sheet locks the corrected vorton ontology after the recent research drift.

It is intended to govern future AMS work on vortons, ordered darkness, magnetism, electricity, light, and topology-first propagation.

## Locked Stack

The AMS stack is:

```text
T0: continuous substrate
T1: stable primary topology
T2: secondary material regimes
```

No modelling mesh, graph, cell complex, lattice, node set, or coordinate scaffold belongs inside this stack as ontology.

## Mechanism Requirement

AMS ontology may be revised, but it must always preserve a candidate mechanism by which substrate organisation can form into perceived matter.

The minimum formation chain is:

```text
continuous substrate
-> local organised disturbance
-> torsion/tension/alignment expression
-> candidate closure-pattern
-> stable primary topology/vorton
-> coupled vorton regimes
-> perceived matter
```

Any future ontology revision must still answer:

- what changes in the substrate
- how the change is retained
- how it returns through itself
- how it persists under perturbation
- how it couples to other stable closures
- how coupled closures become material regimes
- why the resulting regime can be perceived as matter

## `T0`: Continuous Substrate

`T0` is the continuous substrate domain.

It is not:

- material stuff in the ordinary sense
- a field
- energy
- information
- a collection of cells
- a graph
- a lattice
- a mesh
- a collection of hidden pellets

Permitted language:

- continuous substrate
- substrate region
- local substrate state
- local substrate relation
- local substrate constraint
- route through the substrate
- ordered non-expressive substrate ground

Avoid unless explicitly marked as modelling notation:

- substrate cell
- substrate node
- substrate edge
- substrate lattice
- substrate mesh
- substrate graph

## `T1`: Stable Primary Topology

`T1` is stable primary topology in the continuous substrate.

Vortons belong to `T1`.

`T1` is not a layer of small material pieces between substrate and matter. It is the first level at which stable topological identity is present.

Permitted language:

- stable primary topology
- primary topological identity
- stable closure-pattern
- bounded persistence pattern
- invariant closure profile
- torsional knot-state
- returnable constraint organisation

## `T2`: Secondary Material Regimes

`T2` is secondary topology built from stable primary topology, transient structures, and boundary conditions.

Examples include:

- lattices
- materials
- interfaces
- phase boundaries
- magnetic domains
- conductive paths
- dielectric barriers
- engineered field-extended systems

`T2` regimes are real. They are not more fundamental than vortons.

## Locked Vorton Definition

A vorton is:

```text
a stable primary topological closure-pattern in the continuous substrate
```

Expanded:

```text
a bounded, distributed, returnable, coherent pattern of substrate organisation
whose invariant closure profile persists through perturbation
strongly enough to function as primary topological identity
```

## A Vorton Is Not

A vorton is not:

- a material particle in the ordinary sense
- a cell of substrate
- a piece of substrate-stuff
- a bead on a hidden lattice
- a field excitation by itself
- every twist in the substrate
- every transient loop
- every light-like propagation
- every local alignment mismatch
- every magnetic domain
- every electrical route event
- every defect
- every stable `T2` structure

## Minimum Vorton Gate

A candidate may be called a stable vorton only if it satisfies the minimum gate:

```text
B_in(V) > B_min
D_spread(V) > D_min
R_return(V) > R_min
Q_coh(V) > Q_min
L_leak(V) < L_max
X_frag(V) < X_max
I_profile(V,t) preserved across N_min perturbation cycles
```

Where:

- `B_in` = inward boundedness
- `D_spread` = distributedness of retained constraint
- `R_return` = returnability
- `Q_coh` = coherence
- `L_leak` = leakage tendency
- `X_frag` = fragmentation tendency
- `I_profile` = discriminating invariant profile

This gate is not the final mathematics. It is the minimum discipline required before vorton language is allowed.

## Proto-Vorton Definition

A proto-vorton is:

```text
a candidate closure-pattern that has crossed some formation thresholds
but has not yet demonstrated stable primary identity
```

Proto-vorton language is provisional. It must not become a label for any interesting transient.

## Ordered Darkness

Ordered darkness is:

```text
the coherent non-expressive substrate ground
```

It is not:

- nothingness
- a substance separate from the substrate
- a causal agent
- a vorton source-object
- a mystical reservoir

Do not write:

```text
darkness generates vortons
```

Write:

```text
ordered darkness is the coherent non-expressive substrate ground
within which stable closure can form under suitable torsion, tension,
alignment, closure, and threshold conditions
```

## Modelling Notation Rule

Discrete notation is permitted only as representation.

Acceptable:

```text
For modelling, represent the continuous substrate by local regions R_i
and adjacency relations A_ij.
```

Not acceptable as ontology:

```text
The substrate is made of cells connected by edges.
```

Preferred notation:

```text
S       continuous substrate domain
R_i     local modelling region within S
A_ij    adjacency relation between modelling regions
X(x,t)  local substrate state
J(x,t)  local constraint routing
V       candidate vorton region
dV      operational boundary of candidate region
```

## Relation To Light

Light is transient coherent propagation.

A vorton is stable primary closure.

Light-like propagation may interact with vortons, be absorbed by material regimes, or contribute to threshold stress under special conditions. It should not be described as a vorton unless the minimum vorton gate is met.

## Relation To Electricity

Electricity is ordered route slip, occupancy reseating, or stored mismatch along admissible paths.

Vortons may constrain or support route-bearing material identity, but ordinary current is not vorton motion and not vorton generation.

## Relation To Magnetism

Magnetism is alignment-closure mismatch and its propagation through matter and substrate routes.

Vortons may anchor, bias, resist, or permit alignment changes, but a magnetic domain is not automatically a vorton.

## Failure-First Requirement

Every future vorton-generation claim must first explain why the candidate does not fail as:

- diffuse topology
- collapse-prone topology
- overconstrained topology
- underreturned topology
- unstable oscillatory topology
- incompatible adjacency topology
- threshold-fragile topology
- ordinary transient propagation
- secondary material structure

If the failure modes are not addressed, the candidate should not be called a vorton.

## Research Prompt Rule

Future research prompts should include:

```text
Do not treat the substrate as made of cells, nodes, pellets, grains, or lattice units.
If discrete notation is used, explicitly mark it as modelling representation only.
Preserve the T0/T1/T2 distinction.
Define vorton as stable primary topology, not as particle or substrate unit.
Start with failure modes before assigning vorton identity.
Do not collapse light, magnetism, electricity, or darkness into vorton generation.
Frame all claims as coherence-building, not direct proof of invisible substrate.
```

## Status

This definition sheet supersedes any supporting research language that treats cells, graphs, lattices, or meshes as substrate ontology.
