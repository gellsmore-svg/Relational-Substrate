# AMS Magnetism as Alignment-Constraint Propagation v1

## Purpose

This note explains the phrase:

```text
magnetism as alignment-constraint propagation
```

The aim is to make the idea graspable without turning magnetism into material substance or treating vortons as tiny magnets.

## Starting Point

In AMS, magnetism should not be treated as:

```text
a material fluid
a separate invisible stuff
a vorton by itself
```

The cleaner account is:

```text
magnetism is a mode of alignment constraint in vorton-bearing secondary regimes
and its propagation is the handoff of alignment mismatch through admissible routes.
```

## What Propagation Means Here

`Propagation` does not mean a lump travels from A to B.

It means:

```text
a local alignment condition changes,
neighbouring regions must now resolve compatibility,
that resolution changes the next local condition,
and the constraint handoff continues.
```

In plain terms:

```text
one part of the ordered regime is forced to re-answer the question:
"how must I align now?"
```

Then adjacent parts must answer the same question.

That sequential re-answering is propagation.

## Alignment Constraint

An alignment constraint is a rule-like pressure on orientation:

```text
this closure prefers this orientation relative to that closure
```

or:

```text
this region is no longer compatible with its neighbour's orientation,
so one or both must reseat, resist, return, or fail.
```

This is why magnetism is close to vorton language without being identical to vorton identity.

Vortons provide stable primary topology. Magnetic behaviour is one way coupled vorton regimes can coordinate or resist orientation.

## Why A Vorton Can Constrain Passage

The user question is important:

```text
if there is no ordinary matter there, what stops something passing through?
```

The AMS answer is not:

```text
matter blocks it
```

At the primary-topology level, the answer is:

```text
the closure-pattern has admissibility conditions.
```

A transient can pass only if its route structure is compatible with the closure-pattern's boundedness, returnability, orientation, and reseating tolerances.

So the constraining property is not ordinary matter. It is:

```text
stable topological order with admissibility rules.
```

Magnetism is one major expression of those admissibility rules when the relevant issue is orientation/alignment.

## Is The Constraint Magnetism Itself?

Sometimes, but not always.

Better:

```text
magnetism is not the only reason a vorton can constrain transmission.
```

A vorton candidate may constrain by:

- closure incompatibility
- boundedness
- return-path occupation
- orientation mismatch
- route-storage behaviour
- reseating limits
- leakage and fragmentation thresholds

Magnetism is specifically the alignment/orientation aspect of constraint propagation.

So:

```text
all magnetic constraint is admissibility-related,
but not all admissibility constraint is magnetic.
```

## Simple Walkthrough

### Step 1: Stable Alignment

```text
region A and region B are compatible
```

Their closure-patterns have an acceptable orientation relation.

### Step 2: Boundary Bias Arrives

A source, neighbouring regime, or disturbance changes the preferred alignment at region A.

```text
A is now under alignment mismatch
```

### Step 3: A Reseats Or Resists

Region A may:

- reseat
- resist
- store stress
- return the disturbance
- fail

### Step 4: Neighbouring Compatibility Changes

If A reseats, B now faces a changed neighbour.

```text
B must resolve compatibility with A
```

### Step 5: Handoff Continues

The need to resolve alignment passes onward.

```text
A changes -> B must answer -> C must answer -> D must answer
```

This is alignment-constraint propagation.

## Relation To Magnetic Domains

A magnetic domain is not automatically a vorton.

It is a `T2` alignment regime:

```text
many stable primary identities coordinated into a larger orientation pattern
```

When domain walls move, AMS can read this as:

```text
the boundary between two alignment regimes is reseating through the material
```

Again, no lump of magnetism needs to move. The alignment constraint is being re-resolved.

## Relation To Light-Like Propagation

If a light-like transient encounters a magnetic/alignment-bearing regime, several things may happen:

- it passes if its route is compatible
- it delays if alignment affects route timing
- it changes polarisation-like behaviour if orientation matters
- it reflects if boundary compatibility is poor
- it scatters if local alignment is incoherent
- it is absorbed if it couples into material modes

This is why magnetism may constrain propagation without being the whole cause of obstruction.

## Visual Mental Model

Imagine not a particle moving, but a line of dancers changing posture.

One dancer changes orientation. The next must adjust to remain compatible. Then the next. The visible motion travels down the line, but no object has travelled from the first dancer to the last.

For book-facing use, a less human metaphor may be better:

```text
a fold pattern in a flexible sheet being re-tensioned from one edge
```

The change travels because local constraints must be re-satisfied.

## Current Definition

AMS should define magnetic propagation as:

```text
the sequential re-resolution of alignment compatibility through a vorton-bearing
secondary regime, where local closure-patterns reseat, resist, store,
return, or fail under changed orientation constraint.
```

Shorter:

```text
magnetic propagation is alignment mismatch being handed onward through admissible closure relations.
```

## Book-Facing Point

This needs clearer prose in the technical companion.

The book should say that magnetism is not a separate material fluid and not identical with vorton identity. It is an alignment grammar of stable and secondary topology.
