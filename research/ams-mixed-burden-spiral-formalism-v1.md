# AMS Mixed Burden Spiral Formalism v1

## Purpose

This file formalises the mixed burden spiral as a distinct severe ecology pattern.

The recent ecology scenario work suggested that many serious living failures are not well described as:

- pure acute collapse;
- pure chronic degradation;
- or pure containment loss alone.

Instead, they often appear as:

```text
mixed recovery-failure spirals
```

in which:

- multiple machine burdens interact;
- recovery attempts remain real but weak;
- reserve is repeatedly spent;
- synchronisation stays high;
- collapse pressure and cumulative decay feed one another.

This file formalises that pattern directly.

## Central Claim

The central claim is:

> A mixed burden spiral arises when acute burden, cumulative burden, synchronisation, and weakened support interact in such a way that recovery attempts remain locally real but globally insufficient, causing reserve to be spent faster than recoverable coherence is restored. The result is not simple immediate collapse, but a self-worsening oscillatory path in which the ecology repeatedly approaches recovery without regaining stable dominance of the positive loop.

More briefly:

```text
The system keeps trying to recover, but every try costs more future than it restores.
```

## Why the Mixed Spiral Needs Its Own Model

The mixed spiral is not just a midpoint between:

- acute collapse

and:

- chronic decline

It has its own logic.

Why:

- acute burdens remain real;
- cumulative burdens remain real;
- support has not fully vanished;
- recovery still occurs in partial episodes;
- but those episodes no longer restore stable headroom.

That makes the mixed spiral especially important for the life boundary because it explains why a system can:

- still act;
- still repair;
- still compensate;

and yet be moving toward death.

## Mixed Spiral Definition

The mixed burden spiral is the ecology state in which:

```text
positive loop activity persists
but negative loop reinforcement dominates over time
```

This means:

- the recovery loop is not absent;
- the recovery loop is no longer winning;
- each recovery cycle fails to restore enough reserve and support to re-establish low synchrony and low cascade risk.

This is the decisive pattern.

## Core Variable Signature

The strongest current mixed-spiral signature is:

```text
R_cap   = low-moderate and unstable
R_res   = low and progressively depleted
C_sup   = moderate-low and weakening
B_sync  = high and persistent
F_casc  = high and recurrent
V_E     = near θ_rec with oscillation or intermittent threshold crossing
```

This is the strongest current comparative fingerprint.

## The Defining Oscillatory Structure

The mixed spiral should be understood as a repeated cycle:

```text
burden spike -> partial compensation -> incomplete recovery -> lower reserve baseline -> renewed burden spike
```

That is the core pattern.

Each cycle may:

- temporarily improve local function;
- reduce a visible symptom;
- restore some local coherence;

without actually restoring:

- reserve depth;
- effective support;
- low synchrony;
- durable ecological viability.

This is why the pattern is so dangerous.

## First Formal Loop Representation

The two major loops remain:

## Positive Loop

```text
R_res -> C_sup -> R_cap -> lower B_sync -> lower F_casc -> preserved R_res
```

## Negative Loop

```text
B_sync -> F_casc -> lower C_sup and lower R_cap -> lower R_res -> higher B_sync
```

The mixed spiral is the regime in which:

```text
positive loop remains active locally
negative loop dominates globally
```

This can be represented by:

```text
L_pos > 0
L_neg > 0
but L_neg > L_pos over sufficient time intervals
```

where:

- `L_pos` = effective positive-loop strength
- `L_neg` = effective negative-loop strength

This is the simplest useful formal distinction.

## Spiral Condition

The best first spiral criterion is:

```text
Spiral_E = 1 if:
1. V_E remains near θ_rec or crosses it intermittently
2. L_pos remains non-zero
3. L_neg persistently exceeds L_pos in cumulative effect
4. R_res declines cycle-to-cycle
```

This gives a cleaner condition than:

```text
system looks unstable
```

It states that the system is spiralling when:

- recovery is real;
- but not sufficient;
- and reserve is being irreversibly eaten.

## Reserve Baseline Drift

One of the most important spiral features is:

```text
reserve baseline drift downward
```

Let:

```text
R_res(t_n)
```

be reserve at the end of cycle `n`.

Then the strongest current spiral signature is:

```text
R_res(t_n+1) < R_res(t_n)
```

despite local partial recovery episodes.

This is one of the sharpest formal markers in the whole model.

It means:

```text
the ecology is recovering locally while losing future recoverability globally
```

## Support Hollowing

The second key spiral marker is:

```text
support hollowing
```

This means:

- structural support routes remain;
- effective support weakens cycle by cycle;
- support is increasingly consumed by keeping the present afloat rather than restoring the future.

Let:

```text
C_eff(t_n+1) <= C_eff(t_n)
```

across cycles unless major restoration succeeds.

That gives a second spiral signature:

```text
support remains present but progressively less restorative
```

## Synchronisation Persistence

The third key marker is:

```text
synchronisation persistence
```

In a recoverable acute shock, synchrony spikes and then falls.

In a mixed spiral:

- synchrony may fluctuate;
- but it does not settle back into a low regime.

Formally:

```text
mean(B_sync over cycle window) remains high
```

or:

```text
B_sync oscillates but with no stable return to low synchrony band
```

This is what makes the ecology progressively harder to save.

## Spiral Archetypes

The mixed spiral itself can still appear in several subtypes.

## 1. Energy-Renewal Spiral

Pattern:

- respiratory weakness lowers reserve;
- translation debt rises;
- renewal weakness then worsens respiratory maintenance;
- repeated partial recoveries fail to restore reserve.

This is probably the main biological spiral archetype in the current corpus.

## 2. Boundary-Damage Spiral

Pattern:

- boundary weakness spreads burden;
- damage burden rises;
- repair support is consumed;
- restored local function still leaves support thinner and synchrony higher.

This is the clearest containment-linked spiral.

## 3. Repair-Exhaustion Spiral

Pattern:

- repair remains active;
- but each cycle consumes reserve and support;
- accumulated debt rises faster than restoration;
- the ecology looks actively compensatory even while losing viability.

This is one of the most deceptive severe states.

## 4. Coordination-Failure Spiral

Pattern:

- support routes exist;
- but they are mistimed or misallocated;
- burden is redistributed inefficiently;
- local recoveries create downstream burden elsewhere.

This is the clearest spiral in which intelligence of compensation matters as much as amount.

## Best Current Formal Rules

The mixed-spiral formalism is most useful when expressed as rules.

## Rule 1

```text
A mixed spiral exists when recovery is real but no longer restorative enough to rebuild reserve and support baseline.
```

## Rule 2

```text
The clearest spiral marker is repeated partial recovery accompanied by falling reserve baseline.
```

## Rule 3

```text
Support hollowing is one of the strongest signs that visible compensation is no longer equivalent to owned recoverability.
```

## Rule 4

```text
Persistent high synchrony distinguishes mixed spirals from recoverable acute crises.
```

## Rule 5

```text
Death approach is often not immediate collapse but the point at which repeated recovery cycles become net ecology loss.
```

This is one of the strongest current death-approach lines in the corpus.

## What This Clarifies About the Life Boundary

The mixed spiral strengthens the life boundary significantly.

It explains why a system may still:

- function;
- compensate;
- repair;
- respond;

while:

```text
owned recoverability is nearly lost
```

This matters because it blocks a shallow inference:

```text
recovery activity is present
therefore the system is safely recoverable
```

That inference is false.

The better line is:

```text
recovery is real only if it restores the ecology strongly enough to lower synchrony and rebuild reserve/support baseline
```

That is a major improvement.

## What This Clarifies About Death Approach

The strongest current mixed-spiral reading of death approach is:

```text
death approach often begins when recovery cycles stop restoring future recoverability and instead become part of the system's net decline
```

This is stronger than:

- high burden alone;
- failed function alone;
- visible repair failure alone.

It is a deeper systems claim.
